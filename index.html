Project 3: Visual Recipes

By David Nyugen, Ben Saupp, and Garen Dorsey

Enter a type of food in the above search box. The first API will display the date and time of your search. The second API will gather that type of food and display images and names of the dishes. If you click on the button with the name of the dish, the third API should send you to a recipie link. If it doesn't, that is due to spoonacular's daily limit for requests, or because there is no recipe for that dish.
